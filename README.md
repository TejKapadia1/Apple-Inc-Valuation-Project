# Apple-Inc-Valuation-Project
Financial valuation analysis of Apple Inc. using DCF and Comparable Company methods. Includes cash flow forecasting, WACC estimation, sensitivity analysis, and valuation dashboards built in Excel.

This project presents a comprehensive valuation analysis of Apple Inc. using both Discounted Cash Flow (DCF) and Comparable Company (Relative Valuation) methods. The objective of the project is to estimate the intrinsic value of Apple and compare it with its current market price to determine whether the stock is fairly valued.

The analysis begins with the collection and examination of Apple’s historical financial data, including revenue, operating income, capital expenditures, and working capital changes. Using these financial metrics, Free Cash Flow to the Firm (FCFF) is calculated and projected over a five-year forecast period based on assumptions related to revenue growth, operating margins, and reinvestment requirements.

The projected cash flows are discounted using the company’s Weighted Average Cost of Capital (WACC) to estimate the present value of future cash flows. A terminal value is calculated to capture the long-term value of the company beyond the explicit forecast period. Sensitivity analysis is performed to evaluate how changes in key assumptions such as WACC and terminal growth rate impact the valuation results.

In addition to intrinsic valuation, the project incorporates Comparable Company Analysis by benchmarking Apple against major technology peers such as Microsoft, Alphabet, and Meta. Key valuation multiples including EV/Revenue, EV/EBITDA, Price-to-Earnings (P/E), and Price-to-Book (P/B) are used to estimate Apple’s implied equity value based on market comparables.
The project also includes financial visualization and valuation dashboards to present key insights, including cash flow projections, enterprise value composition, sensitivity analysis, and comparable valuation metrics. Overall, the analysis demonstrates how fundamental valuation techniques and market-based comparisons can be used together to estimate a company’s intrinsic value and support informed investment decisions.
